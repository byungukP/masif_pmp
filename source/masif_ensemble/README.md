### source/masif_ppi_ensemble
Contains entry functions to for MaSIF-ensemble, including for training and evaluation.

+ *masif_ensemble_train.py*: Entry function to train MaSIF-ensemble
+ *masif_ensemble_predict.py*: Entry function to evaluate a protein using MaSIF-ensemble.
+ *masif_ensemble_label_surface.py*: Program to color a ply file by the MaSIF-site predicted score.
